Présentation

Italk-Engine est un moteur de messagerie instantanée modulaire, écrit en Python.
Il gère utilisateurs, groupes, messages et extensions/plugins pour enrichir vos applications.

Installation rapide

git clone https://github.com/jallow-2693/Italk-Engine.git
cd Italk-Engine
python app.py

Fonctionnalités principales

  • Gestion d’utilisateurs et groupes
  • Messagerie instantanée (envoi, réception, horodatage)
  • Système d’événements (hooks)
  • Persistance de l’état
  • Extensions dynamiques (plugins)
  • Configuration JSON centralisée

Intégration dans votre projet

from Engine.core import ItalkEngine

engine = ItalkEngine(config_path="extensions/config.json")
user = engine.connect_user("123", "Alice")
engine.send_message("123", "Hello world!")

Sécurité & tokens

  • L’authentification (par token/JWT) doit être gérée côté API ou serveur
  • Il suffit ensuite d’appeler connect_user() avec les données vérifiées
  • Des informations supplémentaires peuvent être stockées dans metadata

Extensions

Ajoutez ou développez des extensions dans le dossier extensions/.
Pour activer une extension, ajoutez son nom dans extensions/config.json.

Déploiement sur serveur

  • Lancez python app.py sur votre VPS ou machine
  • Intégrez-le à une API REST (FastAPI, Flask, etc.) pour une messagerie web/mobile
  • Hébergez ce site statique sur GitHub Pages pour la documentation
